# Elysium's dotfiles for Hyprland

### These dotfiles are specifically made for my use and i use github just for backup purpose, i don't guarantee they'll work for you out of the box and i don't intend to.

### Features
<ul>
<li> Hyprland – Wayland compositor
<li> Quickshell – Status bar with some additional features as well
<li> Rofi – Application launcher (For now)
<li> Rofi - Wallpaper selector with custom script
<li> Dunst – Notification daemon
<li> Alacritty – Terminal emulator with custom color schemes
<li> Neovim – Configured with lazy.nvim and plugins
<li> Fastfetch – System info
</ul>
